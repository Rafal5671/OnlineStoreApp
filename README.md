## Description

OnlineStoreApp is a modern e‑commerce web application. It allows users to register, log in, browse products, and manage a shopping cart. The frontend is built with **React** and **Material UI (MUI)** for a clean, responsive design. The backend is powered by **Spring Boot** and securely handles user data, including password hashing.

This project demonstrates full‑stack development with modern frontend frameworks, secure backend practices, and RESTful API communication.

---

## Features

* User registration and login with password hashing
* Browse and view products using Material UI components
* Add items to a shopping cart
* Interactive frontend with React state management
* Secure backend APIs with Spring Boot
* Responsive and modern UI

---

## Technology Stack

* **Frontend:** React, Material UI (MUI), JavaScript
* **Backend:** Spring Boot (Java), password hashing
* REST API endpoints for frontend–backend communication
* CSS for custom styling
